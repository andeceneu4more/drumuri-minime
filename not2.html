Metode de reprezentare
Matricea de adiacenta este cea mai simpla forma de reprezentare. Pentru un graf care are n noduri, matricea de adiacenta este un tablou bidimensional cu n linii si n coloane unde pentru orice i si j ( 1 ≤ i, j ≤ n ):
- unde ω(i, j) reprezinta costul arcului (i, j). Datorita orientarii, asa cum am mai spus, arcul (i, j) nu este acelasi lucru cu (j, i). Prin urmare A[i][j] ≠ A[j][i], iar matricea este asimetrica fata de diagonala principala.
In cazul unui graf care nu este ponderat, matricea de adiacenta este definita mult mai simplu: A[i][j] este 1 daca exista arcul (i,j) sau 0, altfel. Spre exemplu pentru graful de mai jos:
