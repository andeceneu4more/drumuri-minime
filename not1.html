Intr-o problema de drum minim este dar un graf orientat G = ( V, E ) si functia de cost
ω : E → ℝ .

Grafuri

Un graf orientat G este o pereche de multimi ( V, E ) , unde V este o multime nevida si
finita de elemente numite varfuri, iar E este o multime de perechi ordonate formate din elemente
distincte din multimea V, numite arce.


De exemplu, pentru graful din prima figura:
V = {1, 2, 3, 4, 5}
E = { (5,3), (1,2), (1,3), (1,4), (1,5), (2,3), (2,4), (2,5), (3,4), (4,5) }

Observatii:
  • Fiecare arc este de forma x = (u,v), unde u se numeste extremitatea initiala, iar v extremitatea finala a arcului. Cu ale cuvinte:"arcul iese din nodul u si intra in nodul v". Se mai spune ca nodurile u si v sunt adiacente, iar arcul x si varful u sunt incidente; varful v se numeste succesor al lui u, iar varful u predecesor al nodului v.
  • Arcul (u,v) este diferit de (v,u)
  • Prin notiunea de perechi ordonate nu trebuie sa intelegem ca un arc este mai mare decat altul, ci pur si simplu ca facem deosebire intre un arc de forma (u,v) si un altul de forma (v,u).
Un graf neorientat G este o pereche de multimi ( V, E ) , unde V este o multime nevida sifinita de elemente numite noduri, iar E este o multime de perechi neordonate formate din elemente distincte din multimea V, numite muchii. Spre deosebire de graful orientat, la nivelul celui neorientat muchia (v,u) coincide cu (u,v). Prin urmare, o muchie are 2 extremitati (niciuna dintre ele nu se numeste finala sau initiala).

De exemplu, pentru graful din a doua figura:
V = {1, 2, 3, 4, 5}
E = { (5,3), (1,2), (1,4), (1,5), (2,3), (2,4), (4,5) }

Fie graful G = ( V, E ). Un subgraf G1 = ( U, F ), unde U ⊂ V si F ⊂ E este un graf obtinut din G prin eliminarea unor varfuri/noduri si a tuturor muchiilor/arcelor incidente lor.
Un graf partial G2 = ( V, F ), unde F ⊂ E este un graf obtinut din G prin eliminarea doar a unor muchii/arce.
